# Библиотека книг

Приложение для управления личной библиотекой книг с графическим интерфейсом на Tkinter.

## Возможности

Добавление книг (автор, название, год, статус прочтения)
Просмотр всех книг в библиотеке
Поиск книг по автору, названию или году
Фильтрация книг по статусу
Обновление статуса книги
Удаление книг
Статистика библиотеки
Хранение данных в JSON файле

## Требования

Python 3.8+
pytest (для запуска тестов)

## User Stories
1. Как пользователь, я хочу добавлять книги в библиотеку, чтобы хранить их в одном месте
2. Как пользователь, я хочу видеть все свои книги, чтобы иметь обзор коллекции
3. Как пользователь, я хочу искать книги по названию, автору или году, чтобы быстро находить нужную
4. Как пользователь, я хочу фильтровать книги по статусу прочтения, чтобы видеть только прочитанные или непрочитанные
5. Как пользователь, я хочу менять статус книги, чтобы отмечать прогресс чтения
6. Как пользователь, я хочу удалять книги, чтобы убирать ненужные
7. Как пользователь, я хочу видеть статистику библиотеки, чтобы оценивать свой прогресс
